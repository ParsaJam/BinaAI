## شرح کوتاه ایده

**بینا‌آی** یک پلتفرم تجارت الکترونیک تخصصی در حوزه فروش عینک است که سه قابلیت منحصربه‌فرد را در یک بستر یکپارچه ارائه می‌دهد:

1. **اسکن هوشمند چهره:** کاربر یک عکس از صورت خود آپلود می‌کند؛ الگوریتم هوش مصنوعی فرم هندسی چهره (بیضی، گرد، مربعی، کشیده) را تشخیص داده و فریم‌های مناسب را پیشنهاد می‌دهد.
2. **تسویه آنلاین بیمه:** برای خریداران عینک طبی، سهمیه درمانی از سازمان‌های بیمه به‌صورت آنلاین استعلام و از فاکتور نهایی کسر می‌شود.
3. **اتصال مستقیم به کارگاه ساخت:** سفارش بدون واسطه به کارگاه‌های تراش عدسی CNC ارسال می‌شود و هزینه نهایی تا ۵۰٪ کاهش می‌یابد.

## بخش‌هایی که با کمک هوش مصنوعی تولید شده‌اند

**تمام کدنویسی این پروژه با استفاده از ابزار هوش مصنوعی (Claude AI — Anthropic and Gemini) انجام شده است.**

دانشجو هیچ کد HTML، CSS یا JavaScript ای به‌صورت دستی ننوشته است. نقش دانشجو در این پروژه شامل موارد زیر بوده است:

- **طراحی ایده اصلی:** ایده کسب‌وکار، مدل درآمدی، و تحلیل‌های کسب‌وکاری توسط دانشجو طراحی شده‌اند.
- **تهیه محتوای تمرین اول:** تمام متون تحلیلی (بلوغ ایده، ارزش‌آفرینی، واسطه‌گری) توسط دانشجو نوشته شده‌اند.
- **راهنمایی و prompt نویسی:** دستورالعمل‌های دقیق برای تولید کد به هوش مصنوعی داده شده‌اند.
- **بررسی و تأیید خروجی‌ها:** دانشجو تمام بخش‌های تولیدشده را بررسی کرده و در صورت نیاز اصلاح درخواست کرده است.
- **یکپارچه‌سازی:** اطمینان از انسجام محتوا و ارتباط منطقی بین بخش‌های مختلف.

## محدودیت‌های نسخه فعلی

- **دمو شبیه‌سازی‌شده است:** اسکن چهره واقعی نیست؛ سیستم فرم صورت را بر اساس قوانین ساده‌ای شبیه‌سازی می‌کند.
- **پرداخت واقعی وجود ندارد:** فرآیند خرید تا مرحله «ثبت سفارش» شبیه‌سازی است.
- **API بیمه واقعی وصل نیست:** استعلام بیمه در دمو با داده‌های از پیش تعریف‌شده انجام می‌شود.
- **بانک اطلاعاتی ندارد:** تمام داده‌ها در JavaScript (حافظه مرورگر) نگهداری می‌شوند.
- **فقط یک صفحه HTML:** سایت به‌صورت SPA (Single Page Application) با JavaScript طراحی شده.
